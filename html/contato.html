DESENVOLVEDOR DO PROJETO

Criando experiências digitais através de ideias.

O RefriClub nasceu como um projeto de estudo e evolução, unindo design, programação e criatividade para transformar uma simples ideia em uma experiência completa na web.

Sobre a criação

Este projeto foi desenvolvido com o objetivo de criar uma plataforma moderna para apresentação de uma marca fictícia de refrigerantes.

A ideia principal foi unir uma interface elegante, animações suaves e interação com o usuário através de tecnologias utilizadas no desenvolvimento web.
